EVO-VISORXML

Lector masivo de XML CFDI 3.3 y 4.0 para analizar tus facturas en segundos.

Visualiza ingresos, gastos, impuestos y tipos de comprobante sin abrir el SAT. Filtra, analiza y exporta todo a CSV en unos cuantos clics.

  • CFDI 4.0
  • XML Parser
  • React + TypeScript

EVOSEC – Network Scanner

Escáner avanzado de redes locales con perfiles Nmap profesionales.

Plataforma para escaneo, diagnóstico y análisis de redes privadas. Construida con TypeScript, Fastify y Nmap. Ideal para monitoreo interno y aprendizaje profesional.

  • Nmap
  • Fastify
  • TypeScript
  • React

EVO-QR – Generador de Códigos QR

Crea códigos QR limpios, rápidos y listos para compartir.

Herramienta desarrollada en HTML, CSS y JavaScript puro para transformar enlaces o texto en códigos QR de alta calidad. Sin dependencias externas, compatible con escritorio y móvil, y diseñada para escalar hacia plantillas, historial, perfiles de usuario y APIs futuras dentro del ecosistema EVORIX.

  • HTML5 + CSS3
  • JavaScript Vanilla
  • Canvas API
  • Exportación PNG
